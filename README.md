Website
=======

Introduction
------------
Blog for xbay users.

Install
-------

    npm install -g hexo-cli
    git clone git@github.com:xbay/website.git
    cd website
    git clone https://github.com/iissnan/hexo-theme-next themes/next
    npm install

Add a blog
----------
Add a markdown file under source/_posts

Preview
-------

    hexo server

Deploy
------

    hexo deploy
